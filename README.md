# Sliday Claude Code Plugins

Claude Code plugin marketplace by [Sliday](https://sliday.com).

## Install

```bash
claude plugins marketplace add sliday/claude-plugins
claude plugins install tamp
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [tamp](https://github.com/sliday/tamp-plugin) | Auto-start Tamp token compression proxy, /tamp:status, /tamp:config |
